# 월급 계산 웹 프로그램

---

### **목적** : 월급 계산
- 직원들의 월급을 손 쉽게 계산하기 위해 프로그램을 구현
- 직원을 등록하고 근무 시간을 입력하면 직원의 월급과 총 인건비를 확인할 수 있다.
